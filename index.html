Organization DMZ

An RFB proxy, written in go that can save and replay FBS files.

This component supports:

  • All modern encodings & most useful pseudo-encodings.
  • Multiple VNC client connections & multi servers (chosen by sessionId).
  • Being a "websockify" proxy (for web clients like NoVnc).
  • Producing FBS files compatible with tightvnc's rfb player (while using tight's default 3Byte color format).
  • Screen Recorder vnc-client and a replay server tho show fbs recordings to connecting clients.
